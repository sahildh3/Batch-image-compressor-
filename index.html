Install ImagePress
Add to your home screen for quick access — works offline too.

One-click install available!
Your browser supports automatic installation.

Must be opened in Safari — Chrome on iOS doesn't support home screen install.
1
Save the image-compress.html file to your iPhone using Files app or AirDrop
2
Open the file in Safari (tap and hold → Open with Safari)
3
Tap the Share button ⬆️ at the bottom of Safari
4
Scroll down and tap "Add to Home Screen" ➕
5
Tap "Add" in the top right — ImagePress will appear on your home screen like a native app!
Android Chrome supports automatic install. Use the button above if it appears, or follow these steps.
1
Open the image-compress.html file in Chrome on Android
2
Tap the three-dot menu ⋮ in the top-right corner of Chrome
3
Tap "Add to Home screen" or "Install app"
4
Tap "Add" to confirm — the app icon appears on your home screen
Tip: For the best experience on Android, serve the file via a local server — see the Desktop tab for how to do this.
If the "Install Now" button appeared above — just click it! Otherwise follow these steps.
1
Open Terminal (Mac/Linux) or Command Prompt (Windows) in the folder containing the HTML file
2
Run: python3 -m http.server 8080
(or python -m http.server 8080 on older Python)
3
Open Chrome or Edge and go to http://localhost:8080/image-compress.html
4
Look for the install icon ⊕ in the address bar, or open the browser menu → "Install ImagePress"
5
Click "Install" — the app opens in its own window, just like a desktop app!
